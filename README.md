## Настройка секретов Firebase

Netlify блокирует билды, если в итоговом бандле встречается шаблон `AIza`.  
Чтобы исключить ложные срабатывания и одновременно не хранить ключ в репозитории:

1. Закодируйте исходный `VITE_FIREBASE_API_KEY` в base64 (`echo -n "AIza..." | base64`).
2. Сохраните результат в переменную окружения `VITE_FIREBASE_API_KEY_BASE64` в Netlify.
3. Не заполняйте `VITE_FIREBASE_API_KEY` в продакшене.  

Приложение автоматически декодирует ключ во время рантайма, поэтому поведение Firebase не меняется.
